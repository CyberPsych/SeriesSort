Welcome to Series Sort.

I created this project to play around with with some designs and methodologies that I was experimenting with. The code is changing as I think of things or get ideas.

Overview

This software is intended to help you organise your series files. You can select a directory and the program will read through all the files in it an decide if they are valid TV series files. If they are valid episodes it adds the files by episode to the respective series. The program can then move them to an organised directory structure.

**Directory Path for Episodes**

Library_Path\Series Name\Season 0n\Series Name S0nE0n.extension

Test Driven Development (TDD)

This project is Test Driven. i.e. Before functionality can be added, a test is created for that functionality. I am using the idea of "Red, Green, Refactor" which can be found at James Shore's site here: http://www.jamesshore.com/Blog/Red-Green-Refactor.html
